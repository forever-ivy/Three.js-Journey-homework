## ğŸš€ Quick Start

### Prerequisites
- Node.js 16+
- npm or yarn

### Running a Project

1. Navigate to any lesson directory:
```bash
cd 04-transform-objects
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm run dev
```

4. Open `http://localhost:5173` in your browser

### Build for Production
```bash
npm run build
```
